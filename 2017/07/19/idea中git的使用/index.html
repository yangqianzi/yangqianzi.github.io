idea中git的使用

Veröffentlicht am 2017-07-19

在git上添加自己的ssh

创建ssh

在gitbash下输入
git config –global user.name “yourname”
git config –global user.email “your email”
ssh-keygen -t rsa -C “your email”

找到ssh

在 C:\Users{user}.ssh 找到 id_rsa.pub
复制里面的ssh

复制进git

登录git
选择profile setting

把ssh复制进git

在idea中添加git

创建本地仓库

创建好项目，选择VCS - > Import into Version Control -> Create Git Repository

然后找块位置放本地仓库

将项目添加进本地仓库

右键项目或选中项目,然后 vcs->git->add

上传项目

vcs->git->commit directory
填写修改信息,commit and push

填写位置信息(从git上面复制下来)
